# 💻 DameOS Terminal Portfolio

A premium, retro-style interactive web terminal built to showcase my projects, skills, and resume. This portfolio ditches the traditional website layout for a sleek, command-line interface that reflects my affinity for Linux, retro computing, and functional design.

## ✨ Features

- **Interactive Command Line:** Fully functional terminal emulator that processes user commands.
- **Boot Sequence:** Authentic startup sequence complete with custom ASCII art.
- **Typewriter Effect:** Dynamic text rendering that simulates typing in real-time.
- **Command History:** Use `Up` and `Down` arrow keys to navigate previously entered commands, just like a real bash terminal.
- **Easter Eggs:** Hidden commands for those who like to poke around (try running `sudo`).

## 🚀 Tech Stack

This project was built entirely from scratch without the use of heavy frameworks or external dependencies:
- **HTML5** for semantic structure
- **CSS3** for styling, retro scanlines, and glow effects
- **Vanilla JavaScript (ES6)** for the terminal logic, state management, and DOM manipulation

## 🏃‍♂️ Running Locally

Since this is a vanilla web project, it doesn't require complex build tools like Node.js or Webpack. You can serve it locally using a simple Python HTTP server.

1. Clone the repository:
   ```bash
   git clone https://github.com/Damian200211/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. Start a local server using Python:
   ```bash
   python3 -m http.server 8000
   ```

3. Open your web browser and navigate to:
   ```
   http://localhost:8000
   ```
